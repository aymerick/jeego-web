jeego-web
=========

Web interface for Jeego


Development
===========

## Installation

* `git clone` this repository
* `npm install`
* `bower install`

## Running

* `ember server`
* Visit your app at <http://localhost:4200>

## Running Tests

* `ember test`
* `ember test --server`

## Building production dist

* `ember build --environment=production`

## Upgrade ember-cli

```
$ npm install ember-cli --save-dev
$ ember init
```

Fix conflicts.
